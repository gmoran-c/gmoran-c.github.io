Project Overview

ComputerAnalyzer is a comprehensive, cross-platform security monitoring and management solution. My vision for this project is to democratize cybersecurity so that any small or mid-sized organization (SME) can protect itself with the same capabilities as a large corporation, without the massive cost of enterprise solutions like Splunk or CrowdStrike.

From a single interface, security teams can monitor local and remote systems, analyze suspicious activity, enforce access policies, and respond to threats in real time. The platform is designed to operate strictly within legal/ethical boundaries, focusing on proportional data collection and compliance (including preparing for GDPR and the EU AI Act).

Current Modules

LogAnalyzer

Defines unauthorized access hours and automatically alerts on cross-platform policy violations via SSH.

UsbAnalyzer

Tracks device activity across platforms, allowing administrators to define strict whitelists of authorized USBs.

Future AI Integration

Roadmap includes AI-powered threat scoring, ImageAnalyzer anomaly detection, and Network Topology Mapping.
